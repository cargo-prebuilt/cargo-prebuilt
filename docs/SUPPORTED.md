## Support Policy

Only the latest stable version is supported.

Alpha, beta, and rc versions are always unsupported.

## Other Notes

Release archives of cargo-prebuilt use
[GitHub's artifact attestations](DOWNLOAD.md#github-attestation)
for versions 0.7.0+.
